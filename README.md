# v7 Direct Writer Web版

PDF上に文字を直接配置し、配置後の文字を選択・編集・削除・移動・折返し幅再編集できるWeb版PDF文字入力ツールです。

## 現在の完成状態

第2系ミッション完了時点で、以下の操作に対応しています。

- PDFアップロード
- PDFプレビュー表示
- プレビュー上のクリック位置への文字追加
- 文字追加時の折返し幅指定
- 配置済み文字の選択
- 選択中 item の個別削除
- 選択中 item の文字内容編集
- 選択中 item の文字サイズ編集
- 文字サイズ入力時の全角数字・全角小数点の自動半角化
- 選択中 item の上下左右微調整
- 微調整ボタンの長押し連続移動
- 長押し移動中の軽量preview処理
- 選択中 item の折返し幅再編集
- 編集内容を反映したPDF出力
- 出力PDFのダウンロード

## 基本操作

1. `PDFを開く` からPDFをアップロードします。
2. プレビュー上で文字を置きたい位置をクリックします。
3. 文字内容と文字サイズを入力します。
4. 折返し右端位置をクリックして文字を配置します。
5. 配置済み文字をクリックすると選択できます。
6. 選択中の文字に対して、編集・削除・微調整・幅編集を行えます。
7. `PDF出力` で編集内容を反映したPDFをダウンロードします。

## 配置済み文字の編集

配置済み文字には内部的に `item_id` が付与され、選択中 item だけを対象に操作します。

- `選択編集`: 文字内容と文字サイズを更新します。
- `幅編集`: 新しい右端位置をクリックし、折返し幅を更新します。
- `選択削除`: 選択中 item だけを削除します。
- `↑` `↓` `←` `→`: 選択中 item の位置を1ptずつ微調整します。
- 微調整ボタン長押し: 連続移動します。

長押し移動中は毎回PDFプレビュー画像を再生成せず、items情報と選択枠だけを軽量更新します。ボタンを離したタイミングで最終位置のpreviewを再生成します。

## ローカル起動

```powershell
pip install -r requirements.txt
python app.py
```

起動後、以下にアクセスします。

```text
http://127.0.0.1:5000
```

## Renderでの起動

Render等では `Procfile` に従い、gunicornで起動します。

```text
gunicorn app:app
```

## ファイル処理と注意

- アップロードPDFは `uploads/` に一時保存されます。
- 出力PDFは `output/` に一時保存されます。
- 古い一時ファイルと古いセッションは一定時間後に削除されます。
- `.env`、仮想環境、PDF出力物、一時ファイルはGit管理対象外です。

## 主なファイル

- `app.py`: Flaskアプリ本体、PDF検証、preview生成、PDF出力、item操作API
- `templates/index.html`: 画面構造
- `static/app.js`: フロント側の状態管理、PDF上のクリック操作、選択・編集操作
- `static/style.css`: 画面スタイル
- `requirements.txt`: Python依存ライブラリ
- `Procfile`: Render/gunicorn起動設定
